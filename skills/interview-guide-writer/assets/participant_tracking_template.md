## Abbreviation Key

**Role**:
- F = Founder/Co-founder
- PM = Product Manager
- D = Designer
- E = Engineer
- O = Other team member

**Stage**:
- PS = Pre-seed
- S = Seed
- A = Series A
- B = Series B+

**Product Type**:
- B2B = Business-to-Business
- B2C = Business-to-Consumer
- Mixed = Hybrid or Platform

**Guide Used** (if using multiple interview guides):
- A = Interview Guide A
- B = Interview Guide B

## Usage Tips

- Fill this out after each interview while details are fresh
- Use abbreviations for quick note-taking during fielding
- Monitor diversity targets in real-time
- Adjust recruitment strategy mid-day if needed (e.g., if all early participants are pre-seed, prioritize seed/Series A in afternoon)
